# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-15

### Added

- Initial release
- `verify` command for payment verification
- `info` command to display supported chains and tokens
- Support for 5 chains: Polygon, Ethereum, Arbitrum, Base, Optimism
- Support for 3 tokens: USDC, USDT, DAI
- JSON and text output formats
- Exit codes for scripting
- BYO-RPC model for custom endpoints
- Time window and block range filtering
- Sender address filtering
- Configurable confirmation requirements
- Amount tolerance for matching
- Rich terminal output with colors and panels
- Comprehensive test suite
